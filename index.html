Directus App Development

Directus is a very useful tool for rapidly developing new applications. However, it can be quite tricky to get started the first time. I’ve created this guide for you to help you through this initial process. I hope you will be able to work on the things that actually matter.

This guide is very much still a work in progress. Check back regularly to see what is going on.
